# PataMatch
O PataMatch é uma aplicação web desenvolvida para conectar pessoas interessadas em adotar animais com aquelas que desejam doar pets para um novo lar.

O projeto simula um sistema real com:

autenticação de usuário
controle de permissões (adotante e doador)
cadastro de animais
demonstração de interesse em adoção

#rodar o projeto 
# Clone o repositório
git clone https://github.com/NayarafMota/Patamatch.git

# Acesse a pasta
cd Patamatch/pataMatch-front

# Abra o index.html
